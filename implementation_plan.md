# VIOLET Website Implementation

## Brand Strategy
- **Colors**: Imperial Purple (#532161) & Queen Pink (#E5D1E5).
- **Typography**: Playfair Display (Serif) for luxury, Montserrat (Sans) for modernism.
- **Imagery**: Mix of real salon interior screenshots and high-fidelity AI-generated service demonstrations.

## Technical Implementation
- **Framework**: Pure HTML5 with Tailwind CSS (via CDN with custom config).
- **Icons**: Lucide Icons & Custom SVG Logo.
- **Interactivity**: Vanilla JS for glassmorphic scroll effects and navigation.

## File Organization
- `/assets`: Contains all images (hero, services, gallery) and raw brand materials.
- `index.html`: Core website file.
- `tailwind.config.js`: Brand token definitions.
